# Glassmorphism Calculator

A sleek, modern web-based calculator featuring a stunning **glassmorphism** design aesthetic. It provides basic arithmetic operations along with a session history tracker and an elegant dark/light mode toggle.

**Live Demo:** [https://d1ff1cult11.github.io/Calculator/](https://d1ff1cult11.github.io/Calculator/)

## Features

- **Glassmorphism UI:** Beautiful, semi-transparent frosted glass design with dynamic background shapes.
- **Dark & Light Modes:** Seamlessly toggle between dark mode (default) and light mode to suit your preferences.
- **Arithmetic Operations:** Perform addition, subtraction, multiplication, and division securely.
- **Calculation History:** Automatically keeps track of your recent calculations. You can also clear the history at any time.
- **Responsive Layout:** Works great on desktop and mobile devices.

## Getting Started

Simply open `index.html` in your favorite modern web browser to start using the calculator! No build steps or installations required.

## Built With

- **HTML5:** Semantic structure.
- **CSS3:** Custom styles, CSS variables, flexbox/grid, animations, and the glassmorphism effect.
- **Vanilla JavaScript:** Logic for calculations, DOM manipulation, theme toggling, and history management.
- **FontAwesome:** Scalable icons for operations and theme toggle.
- **Google Fonts:** Utilizing the modern "Outfit" typography.

## Project Structure

- `index.html` - The main entry point and structural layout.
- `style.css` - Contains all styling rules and theme variables.
- `script.js` - Contains the application logic for the calculator.
